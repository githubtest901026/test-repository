Cloning A Repository

  1. Generate your SSH Key
    • Install Git for Windows
    • Open Git Bash
    • Type pwd and ensure that it looks like it is pointing to your user folder
      • NOTE: You can get to your user folder at any time with cd ~
    • Make the ".ssh" folder with: mkdir .ssh
    • Change your directory with the cd command to .ssh: cd .ssh
    • Run ssh-keygen.exe and leave all options as default (keep pressing enter)
  1. Upload your key to github
    • Open up the file explorer and navigate to your user folder and then the .ssh folder. (C:\Users\YourUserFolder\.ssh)
    • Open id_rsa.pub in notepad.
    • Copy the entire contents of the file (CTRL + A and then CTRL + C)
    • Log into your github account
      • User Icon (top left)
      • Settings
      • SSH and GPG Keys
      • Add Key
      • Name it whatever you want, like Work Laptop or Personal Laptop. Whatever will let you know in the future that that key is for that computer.
  1. Checkout from IntelliJ

    • VCS get from version Control
    • Get the URL of the respository you want to clone: https://github.com/githubtest901026/test-repository
    • Click Clone or Download
    • Copy the Git SSH URL: git@github.com:githubtest901026/test-repository.git
